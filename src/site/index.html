Vert.x 是一个基于 JVM 构建 reactive 应用的工具集.

{% icon "install" %} 下载 v{{ project_version }}

可伸缩性

Vert.x 是 事件驱动 , 无阻塞的. 这意味着你的应用可以使用少量核心线程就能进行大量的并发处理. Vert.x 使你的应用可以使用最少的硬件进行扩展.

  • Java
  • JavaScript
  • Groovy
  • Ruby
  • Ceylon

import io.vertx.core.AbstractVerticle;
public class Server extends AbstractVerticle {
  public void start() {
    vertx.createHttpServer().requestHandler(req -> {
      req.response()
        .putHeader("content-type", "text/plain")
        .end("Hello from Vert.x!");
    }).listen(8080);
  }
}

vertx.createHttpServer()
  .requestHandler(function (req) {
    req.response()
      .putHeader("content-type", "text/plain")
      .end("Hello from Vert.x!");
}).listen(8080);

vertx.createHttpServer().requestHandler({ req ->
  req.response()
    .putHeader("content-type", "text/plain")
    .end("Hello from Vert.x!")
}).listen(8080)

$vertx.create_http_server().request_handler() { |req|
  req.response()
    .put_header("content-type", "text/plain")
    .end("Hello from Vert.x!")
}.listen(8080)

import io.vertx.ceylon.core { ... }
import io.vertx.ceylon.core.http { ... }
shared class Server() extends Verticle() {
  start() => vertx.createHttpServer()
    .requestHandler((HttpServerRequest req) =>
      req.response()
        .putHeader("content-type", "text/plain")
        .end("Hello from Vert.x!")
    ).listen(8080);
}

多语言

你可以使用多种语言编写 Vert.x 应用,包括 Java, JavaScript, Groovy, Ruby, and Ceylon.

Vert.x 不宣扬哪种语言最好. 你可以根据当前的任务和你的团队的技能栈来选择语言.

我们为 Vert.x 支持的每种语言都提供了通用的 APIs.

General purpose

Vert.x is incredibly flexible - whether it's simple network utilities, sophisticated modern web applications, HTTP/REST microservices, high volume event processing or a full blown back-end message-bus application, Vert.x is a great fit.

Vert.x is used by many different companies from real-time gaming to banking and everything in between.

Unopinionated

Vert.x is not a restrictive framework or container and we don't tell you a correct way to write an application. Instead we give you a lot of useful bricks and let you create your app the way you want to.

Need some guidance? We provide a large selection of examples to get you started for the particular type of application you want to write.

Vert.x 是有趣的

享受再次成为开发人员.

不像传统的有约束的应用容器, Vert.x 带来难以置信的力量和敏捷性,让你可以用希望的语言 来创建引人注目的,可扩展的21世纪应用,将困扰最小化.

  • Vert.x 是轻量级的 - Vert.x core 大约 650kB 大小.

  • Vert.x 是快速的. 这里是第三方的 Benchmarks.

  • Vert.x 不是应用服务器. 在你部署的应用不存在作为整体的 Vert.x 实例. 你可以在任何地方运行你的应用.

  • Vert.x 是模块化的 - 当你需要更多的话直接增加就可以了.

  • Vert.x 是简单的但不是过分单纯的. Vert.x 允许你简单地创建强大的应用程序.

  • Vert.x 是创建轻量级,高性能, 微服务的理想选择.
